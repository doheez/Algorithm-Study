## ❗ 풀이

각 문자마다 우선순위를 정해주면 되는 문제이다.<br/>
각 문자의 자리수를 이용해 값이 가장 높은 문자부터 높은 숫자를 부여한다.<br/>
<br/>
예를 들어, `ABC`라는 문자열이 있다면 `A`에는 `10의 2승`, `B`에는 `10의 1승`, `C`에는 `10의 0승`
 value값을 원래 있던 값에 더해준다. 이때 HashMap을 사용한다.<br/>
