## ❗ 풀이

DFS 와 DP 를 함께 사용하는 문제<br>
숫자가 낮은 쪽으로만 이동한다는 것은 한번 탐색한 곳은 다음번에 다른길로 똑같은 곳에 도착했을 때 다음길이 확정이 된다는 것을 의미 --> memoization<br>
