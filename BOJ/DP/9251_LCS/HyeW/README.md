## ❗ 풀이

아래와 같은 두 문자열이 있을때 LCS는 최장 공통 부분 수열(ACAK)의 길이를 찾는 문제이다.<br/>
__ACAYKP__ <br/>
**CAPCAK** <br/>
이중 for문과 이차원 배열을 사용하여 두 문자열을 첫번째부터 마지막까지 하나하나 비교하는 방식이다<br/>
__배열에는 비교하는 두 문자열의 LCS를 계속 채워주는 것이다.__<br/>
두가지의 분기점이 나올 수 있는데<br/>
예를 들어, 문자열 ACA와 CAP를 비교할 때, 각 문자열의 마지막 문자가 A와 P로 같지 않다면<br/>
AC와 CAP의 LCS, ACA와 CA의 LCS중 최대값이 ACA와 CAP의 LCS가 되는 것이다.<br/>
`DP[i][j] = Math.max(DP[i-1][j], DP[i][j-1])`이 된다.<br/>
<br/>
그리고 ACA와 CA와 같은 문자열과 같이 문자열의 마지막이 같다면<br/>
현재 문자를 비교하는 과정 이전 문자열의 LCS에 +1을 해준다.<br/>
AC와 C의 LCS를 구하여 +1해주는 것이다.<br/>
`DP[i][j] = DP[i-1][j-1] + 1`이라 표현할 수 있다.<br/>
