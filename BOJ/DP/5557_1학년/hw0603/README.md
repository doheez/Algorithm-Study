## ❗ 풀이
상근이는 음수와 20보다 큰 수는 알지 못하므로 20x20 DP배열을 미리 선언하여 사용한다.  
`dp[i][j] = i번째 인덱스의 수를 가지고 j를 만들 수 있는 경우의 수`  
  
N-1번째에서 구한 경우의 수에서,
- 다음 숫자를 더했을 때 -> 20이하
- 다음 숫자를 뺐을 때 -> 0이상
이라면, N-1번째 경우의 수에 누적해 주면 N번째의 경우의 수를 구할 수 있다.
