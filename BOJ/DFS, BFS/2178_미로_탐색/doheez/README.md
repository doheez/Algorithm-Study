## ❗ 풀이
bfs로 최단 거리를 탐색한다. maze와 크기가 같은 count 이차원 배열을 둬서 다음 노드를 탐색할 때 이전 노드의 count값에 1을 더해 거리를 계산한다.

노드 방문 표시, 거리 계산, 반복문 종료 조건 걸기를 큐에서 노드를 꺼낸 시점에 하는 게 아니라, 상하좌우로 다음 노드를 탐색할 때 진행해서 시간을 줄이도록 한다.
