## ❗ 풀이
BFS로 도화지를 탐색하면서 방문한 칸은 0으로 바꾸며 카운트를 증가시키고, 상하좌우 4방향 모두 갈 곳이 없으면 현재까지의 카운트를 리턴한다.  
메인 함수에서는 도화지 한 칸 마다 값을 계산하면서 방문하지 않은 칸(값이 1인 칸)에 대해서 `bfs()`를 호출한다.
