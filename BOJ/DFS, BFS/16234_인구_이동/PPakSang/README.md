# Info

문제 출처 : [BOJ 인구이동](https://www.acmicpc.net/problem/16234)

## ❗ 풀이

DFS 알고리즘을 사용한 구현문제에 가까웠음<br>
군집화를 하는데 해당 군집을 어떻게 저장할지에 대한 고민이 필요했다.<br>
나는 Location 객체를 만들어 같은 군집끼리의 위치 정보를 Stack 에 모두 저장하고<br>
탐색이 끝난 후(군집화 종료) Stack 에서 하나씩 꺼내가며 같은 군집끼리 필요한 연산을 수행하였다.<br>
다른 답안을 찾아보니 군집화 종료 후에 visited 에 마킹된 애들만 다시 BFS 하면 되는 것을 깨달았다.<br>
