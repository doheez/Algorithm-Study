## 📘 스터디 규칙 (예시)
1. 매주 월요일 오후 X시, 팀원 회의를 통해 알고리즘 주제를 선정한다.
2. 해당 주제에 대해 매일(주말 제외) 문제를 하나씩 푼다.
3. 매일 사다리 타기를 통해 한 명을 뽑아서 본인 코드 설명을 진행한다.
4. 코드 설명자 이외의 다른 사람들은 코드 리뷰를 한다.
